# Code of Conduct

**🌐 Language:** **English** · [Español](CODE_OF_CONDUCT.es.md)

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
v2.1.

In short: treat everyone with respect. Harassment, insults, and discrimination
are not tolerated. Keep the discussion technical and constructive.

To report unacceptable behavior, contact the maintainer at
**deleonalonso77@gmail.com**. All reports are reviewed confidentially.
