# Código de Conducta

**🌐 Idioma:** [English](CODE_OF_CONDUCT.md) · **Español**

Este proyecto adopta el [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
v2.1.

En resumen: trata a todas las personas con respeto. No se tolera el acoso, los
insultos ni la discriminación. Mantén la discusión técnica y constructiva.

Para reportar comportamiento inaceptable, contacta al mantenedor en
**deleonalonso77@gmail.com**. Todos los reportes se revisan de forma confidencial.
